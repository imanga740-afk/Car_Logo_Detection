# Car Logo Detection

## Overview
This project focuses on detecting and identifying car logos from images using computer vision and deep learning techniques.

## Problem Statement
Automatic car logo detection can be useful in intelligent transport systems, traffic monitoring, and vehicle brand analysis.

## Dataset
- Car logo image dataset
- Source: custom collected images
- Dataset not uploaded due to size limitations

## Methodology
- Image preprocessing
- Model training for logo detection
- Prediction and visualization of detected logos

## Tools & Technologies
- Python
- Flutter for deployment
- YOLO / CNN (specify model used)

## Results
The model is able to detect car logos from images with promising accuracy.

- Increase dataset size
- Improve detection accuracy
